# botfarm

pnpm monorepo for internal chat bots.

## Packages

- `workspace-buddy`: Telegram AI teammate bot for idea critique, planning, and summaries.

## Quick Start

```bash
cd /Users/yurykorotovskikh/repos/unimetrics/botfarm
pnpm install
pnpm dev
```

`pnpm dev` runs `workspace-buddy`.

Package docs: `/Users/yurykorotovskikh/repos/unimetrics/botfarm/workspace-buddy/README.md`
